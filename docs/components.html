Collapsible

You can use the collapsible and collapsed to build animated accordion component

Class Description
collapsible This class sets the max-height of your element with a very high value
collapsed When you apply animate collapsible to an element, toggling collapsed will collapse or expand that element

  • One

    I'm the first child

  • Two

    I'm the second child

  • Three

    I'm the third child
<ul id="collapsible_example">
  <li class="mvn">
    <p class="backgroundLowlight clickable mvn">One</p>
    <div class="animate collapsible collapsed pvl">I'm the first child</div>
  <li class="mvn bvs">
    <p class="backgroundLowlight clickable mvn">Two</p>
    <div class="animate collapsible collapsed pvl">I'm the second child</div>
  <li class="mvn">
    <p class="backgroundLowlight clickable mvn">Three</p>
    <div class="animate collapsible collapsed pvl">I'm the third child</div>
</ul>

  $('#collapsible_example').each(function() {
    var $list = $(this);
    $list.find('li').on('click', function() {
      $list.find('div').addClass('collapsed');
      $(this).find('div').removeClass('collapsed');
    });
  });

Hover

You can use the hover and hoverDisplay classes to display elements on hover

Class Description
hover This class goes on a container element. The element's size and shape will be used for determining when the hover effect is on and off.
hoverDisplay The element to display on hover, this does not need to be a direct child of hover and can used as many times as needed.

Overlay

Occasionally you need some text that shows on top of an image, so in that case use an overlay class.

Overlay Skins

Class Description
overlay Position the element as an overlay (abosolute) with no background color
Skins
overlayHighlight Apply the green color skin
overlayLowlight Apply the black color skin
overlayAction Apply the blue color skin -- designed for actionable/clickable element
overlayMidnight Apply the midnight color skin
overlayDefault Apply the white color skin
Positions
overlayTop Position the overlay at the top
overlayBottom Position the overlay at the bottom
overlayLeft Position the overlay on left
overlayRight Position the overlay on right
overlayMiddle Position the overlay in the middle
overlayFullWidth Make the overlay be full width
overlayFull Make the overlay cover the entire image
Animations Meant to use with animate classes
overlayFadeOut Make the overlay cover fade out to zero opacity.
overlaySlideUp Make the overlay slide 100% to the top
overlaySlideRight Make the overlay slide 100% to the right
overlaySlideDown Make the overlay slide 100% to the bottom
overlaySlideLeft Make the overlay slide 100% to the left

Table

Class Description
table table-bordered Basic table with borders on the cells
table table-primary With header in primary background color; and alternating background color between the rows
Sorting
sort-actions inactive Lowlight color applied to the carets to indicate the column isn't sorted
sort-actions active Highlight color applied to the carets to indicate that the column is sorted

Table Whitespace

Table cells can have their cell padding adjusted by adding a class to table, td or th element. It works similar to the whitespace The classes are constructed by specifying <type><location><size> (see table and example below).

Type Description
t table Affects only the spacing on <th> and <td> elements, you can set this directly on those elements or on the <tbody> or <table> element itself.
Location
a all Adds spacing on all sides of the element
h horizontal Adds spacing to the left and right sides of the element
v vertical Adds spacing to the top and bottom sides of the element
Size
xl extra large Adds an extra large amount of spacing (36px)
l large Adds a large amount of spacing (21px)
m medium Adds a medium amount of spacing (12px)
s small Adds a small amount of spacing (6px)
xs extra small Adds an extra small amount of spacing (3px)
n none Sets the spacing to 0
